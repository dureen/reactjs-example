Installation
cd project-folder
npx create-react-app .

npm install react-bootstrap bootstrap
import 'bootstrap/dist/css/bootstrap.min.css';

Main: `src/index.js`

Add directory and file:\
- `components`\
- `assets`\
- `pages`\
- `style`\

Router: `src/App.js`
```
npm i -D react-router-dom
```
Router Layout: `src/pages/Layout.js`
